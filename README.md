# 📚✨ NotebookLM 互动闪卡 - 极致沉浸版

一款基于 AI 手势识别的沉浸式学习工具，专为深度专注和高效记忆设计。  
支持 **桌面端摄像头手势** 和 **移动端触控手势**，配合 **冥想/雨声双音轨**，让学习回归专注本质。

## ✨ 核心特性

### 🤖 多模态交互
- **桌面端**: 基于 Mediapipe 的 AI 手势识别 (OK手势、双手摊开)
- **移动端**: 原生触控支持 (滑动切题、长按掌握、边缘唤醒)

### 🎯 心流沉浸模式
- 鼠标静止 3 秒自动隐藏 UI
- 全屏模式 + 呼吸引导
- **双音轨系统**: 
  - 初始界面播放冥想音乐 🧘
  - 专注模式切换为下雨白噪音 🌧️

### 📊 智能学习
- 自动跳过已掌握题目
- 顺序/乱序切换
- 进度实时追踪与通关庆祝

## 📖 操作指南

### 桌面端 🖥️

| 操作 | 效果 |
|------|------|
| 点击卡片 / 空格键 | 翻转卡片 |
| 左右方向键 | 切换题目 |
| Enter 键 | 标记已掌握 |
| F11 | 全屏模式 |
| 👌 OK 手势 | 标记已掌握 |
| 👐 双手摊开 | 下一题 |

### 移动端 📱

| 操作 | 效果 |
|------|------|
| 点击卡片 | 翻转卡片 |
| 左右滑动 | 切换题目 |
| 长按卡片 | 标记已掌握 |
| 点击顶/底边缘 | 唤醒界面 |
| 📷 摄像头按钮 | 开启简易手势 (仅OK手势) |

## 🚀 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动服务器**
   ```bash
   npm start
   ```

3. **访问应用**
   浏览器打开 `http://localhost:8081`

## 🛠️ 技术栈

- **前端**: 原生 JavaScript (ES6+), CSS3 Variables
- **AI 模型**: MediaPipe Hands (Google)
- **数学渲染**: MathJax 3.2
- **后端**: Express.js (Node.js)
- **进程管理**: PM2 (24/7 在线)

## 📝 题库格式

支持 standard CSV 格式 (`问题,答案`)，支持 LaTeX 公式 `$E=mc^2$`。
