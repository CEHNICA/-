# 心流闪卡学习系统 📚✨

一款极致沉浸的知识点学习工具，支持手势控制和自动播放模式。

## 功能特点

- 🎯 **心流模式**：沉浸式学习体验，自动隐藏干扰元素
- 📱 **手机友好**：滑动切换题目，点击翻转卡片
- 🎵 **音效反馈**：翻页声效 + 通关音乐
- ✌️ **手势控制**：摄像头识别 OK 手势标记已掌握
- 🔄 **智能跳过**：自动跳过已掌握的题目
- 🏆 **完成庆祝**：全部掌握后播放胜利音乐

## 快速开始

1. 安装依赖：
```bash
npm install
```

2. 启动服务：
```bash
npm start
```

3. 打开浏览器访问：`http://localhost:8081`

## 操作方式

| 操作 | 效果 |
|------|------|
| 点击卡片 / 空格键 | 翻转卡片 |
| 左右滑动 / 方向键 | 切换题目 |
| 按下 Enter | 标记已掌握 |
| 按下 F11 | 全屏 + 播放背景音乐 |

## 技术栈

- 原生 JavaScript (ES6 Modules)
- Express.js 静态服务器
- MediaPipe 手势识别
- MathJax 数学公式渲染
