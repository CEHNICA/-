📚✨ NotebookLM 互动闪卡 - 极致沉浸版
一款基于 AI 手势识别的沉浸式学习工具，专为深度专注和高效记忆设计。通过摄像头手势控制、自动播放模式和心流优化界面，让学习变得更加自然和高效。

✨ 核心特性
🤖 AI 手势控制 - 基于 MediaPipe 的实时手势识别
👌 单手 OK 手势：标记已掌握
👐 双手摊开：翻转卡片/下一题
无需触碰设备，完全解放双手
🎯 心流沉浸模式
鼠标静止 3 秒自动隐藏所有 UI 元素
自动隐藏光标，杜绝视觉干扰
全屏模式 + 冥想背景音乐
呼吸引导动画帮助进入专注状态
📊 智能学习系统
自动跳过已掌握的题目
顺序/乱序切换模式
可自定义正反面停留时间
学习进度实时追踪
完成后播放通关音效和庆祝动画
📐 数学公式支持
集成 MathJax 3.x 完美渲染 LaTeX
支持行内公式和独立公式
自动优化公式渲染性能
🎵 沉浸式音效
翻页音效反馈
冥想背景音乐
通关胜利音乐
可一键开关音效
📱 全平台适配
响应式设计，支持桌面和移动端
移动端支持滑动手势
键盘快捷键支持（空格翻转、方向键切换、Enter 标记）
🛠️ 技术栈
前端: 原生 JavaScript (ES6 Modules)
AI 识别: MediaPipe Hands (Google)
数学渲染: MathJax 3.2.2
服务器: Express.js
字体: Google Fonts (Noto Sans SC + Outfit)
样式: 现代 CSS3 (渐变、动画、glassmorphism)

bash
# 安装依赖
npm install
# 启动服务器
npm start
# 访问应用
# 浏览器打开 http://localhost:8081
📖 使用指南
基础操作

点击卡片 / 空格键：翻转卡片
左右滑动 / 方向键：切换题目
Enter 键：标记已掌握
F11：进入全屏沉浸模式
手势控制

点击摄像头图标开启手势识别
允许浏览器访问摄像头
使用手势进行无接触操作
自动播放

点击播放按钮
在设置中调整停留时间
选择"仅看题目"或"题目+答案"模式
